## 2. `if` とは何か
`if`（イフ）とは、
**「もしこの条件なら、この処理をする」**
という命令です。

日常でも、同じ考え方をしています。

### たとえば
- もし雨なら、傘を持つ
- もし予定が空いていれば、打ち合わせを入れる
- もしスコアが70以上なら、合格にする

プログラミングもこれと同じです。

---

## 3. いちばんシンプルな形
```python
score = 80

if score >= 70:
    print("合格")
```

### このコードの意味
- `score = 80` → score という箱に 80 を入れる
- `if score >= 70:` → もし score が 70以上なら
- `print("合格")` → 合格と表示する

この場合は 80 なので、`合格` が表示されます。

---

## 4. 条件が合わないときはどうなるか
```python
score = 60

if score >= 70:
    print("合格")
```

この場合、60 は 70以上ではないので、
**何も表示されません。**

つまり `if` は、
**条件が合ったときだけ動く**
のがポイントです。

---

## 5. よく使う比較の形
条件では、よく次の記号を使います。

- `==` → 同じ
- `!=` → 違う
- `>` → より大きい
- `<` → より小さい
- `>=` → 以上
- `<=` → 以下

### 例
```python
age = 20

if age >= 18:
    print("成人")
```

---

## 6. `:` とインデントが大事
`if` を使うときは、次の2つが大事です。

### ① 行の最後に `:` をつける
```python
if score >= 70:
```

### ② 次の行を少し下げる（インデント）
```python
if score >= 70:
    print("合格")
```

この空白の下がりがあることで、
**「この処理は if の中です」**
とPythonに伝わります。

---

## 7. `else` もある
`else`（エルス）は、
**「それ以外なら」** という意味です。

```python
score = 60

if score >= 70:
    print("合格")
else:
    print("不合格")
```

この場合は 60 なので、`不合格` が表示されます。

つまり、
- `if` → もし〜なら
- `else` → それ以外なら

です。

---

## 8. 仕事への置き換え例

### 広告運用
```python
cv = 3

if cv >= 5:
    print("反応は悪くない")
else:
    print("改善案を考える")
```

→ 数字によって判断を変えるイメージ

### SNS運用
```python
platform = "X"

if platform == "X":
    print("短く強い文章を意識する")
else:
    print("媒体に合わせて調整する")
```

→ 媒体によって考え方を変えるイメージ

### 営業
```python
client_type = "不動産"

if client_type == "不動産":
    print("事例を入れて提案する")
else:
    print("業界に合わせて提案を変える")
```

→ 相手に応じて提案を出し分けるイメージ
