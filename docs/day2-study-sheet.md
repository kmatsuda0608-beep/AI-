## 2. Pythonとは何か
Python（パイソン）は、
**人が比較的読みやすい形で書けるプログラミング言語**です。

特に、以下のような仕事と相性がいいです。
- AI活用
- 自動化
- データ整理
- 簡単なツール作成

難しく考えすぎなくて大丈夫です。
まずは、
**「AIに指示しながら一緒に使える言語」**
くらいの理解で十分です。

---

## 3. 文字列とは
文字列（もじれつ）とは、
**文字や文章のデータ**です。

たとえば、
- 名前
- 商品名
- SNS投稿文
- 広告タイトル

などは文字列です。

### 例
```python
name = "おまつさん"
post_title = "Meta広告で反応を上げる3つの考え方"
```

`" "` で囲まれているものは、基本的に文字列と考えてOKです。

---

## 4. 数字とは
数字はそのまま、
**数のデータ**です。

たとえば、
- 広告予算
- クリック数
- CV数
- 投稿本数

などです。

### 例
```python
budget = 30000
clicks = 125
cv = 7
```

文字列との違いは、
**数字は計算に使える**ことです。

### 例
```python
today_cv = 3
yesterday_cv = 2
all_cv = today_cv + yesterday_cv
```

この場合、`all_cv` は `5` になります。

---

## 5. 変数とは
変数（へんすう）とは、
**データを入れておく箱**です。

### イメージ
- `name` という箱に名前を入れる
- `budget` という箱に予算を入れる
- `theme` という箱に投稿テーマを入れる

### 例
```python
name = "おまつさん"
budget = 50000
theme = "SNS集客"
```

左側が**箱の名前**、右側が**入れる中身**です。

---

## 6. printとは
`print`（プリント）とは、
**中身を画面に表示する命令**です。

### 例1
```python
print("こんにちは")
```

→ 画面に `こんにちは` と表示されます。

### 例2
```python
name = "おまつさん"
print(name)
```

→ 画面に `おまつさん` と表示されます。

つまり `print` は、
**「この中身を見せて」** という命令です。

---

## 7. 入力と出力の考え方
プログラミングでは、よく
**入力（入れる情報）** と **出力（出てくる結果）** を考えます。

### たとえば
- 入力：広告タイトル
- 出力：画面に表示された広告タイトル

### シンプルな例
```python
title = "夏の集客キャンペーン"
print(title)
```

このコードでは、
- 入力したもの：`"夏の集客キャンペーン"`
- 出力されたもの：画面に表示された `夏の集客キャンペーン`

です。

最初はこの感覚が大事です。
